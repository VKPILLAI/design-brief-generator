# Design Brief Generator
A tool for designers to generate structured, client-ready creative briefs in under two minutes.

**Problem:** Writing briefs from scratch takes 20–30 minutes every time — for the same information, in a different order, every project.

**Solution:** Fill in the form. Get a formatted brief instantly, with project-specific deliverables auto-suggested.
